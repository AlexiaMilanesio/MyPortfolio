The Pokemon App

pokemon world

Henry Bootcamp - Full Stack Solo Project

The first Full Stack App I developed by myself.

Project goals: build an app using React, Redux, NodeJS and Sequelize - PostgreSQL (technologies learned during the bootcamp) to affirm and connect the learned concepts.



With this app you can check out 40 different pokemons from the pokeapi. You can search for pokemons by name, filter and sort them. You can also create your own new pokemon. For the filtering and sorting functionalities, I wasn't allowed to use the external API endpoints that already return the filtered or sorted results, I had to do it myself.
