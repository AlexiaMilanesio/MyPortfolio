Blazar Academy - Final Project

This Battleship Game was design and built by a group of 4 people in 1 week. This was the final project for the Blazar Academy (Milan, Italy).

In this Cooperative Battleship you're able to play with a team, but the team you're playing for is a mistery.

Project goals: build a Cooperative Battleship game using the technologies learned during the Academy, affirm and connect the learned concepts, learn the best practices, learn and practice the GIT Workflow, teamwork and soft skills development.

This game was intended to be played in the same physical place, on the same computer.

It can be played by 2 people. In this case each person will belong to a different "group", just like in the original Battelship game. But it can be played by up to 6 people. In this cooperative version of the game, groups are randomly assigned and each group has at least 1 player.

The team that wins is the team that manages to sink the entire fleet of ships of the opposing team.
